# Student phone install — class handout (Bengali)

Share this with students and parents via WhatsApp, print for class, or project
on screen. One institute link is enough for every student.

---

## Link to share

Ask your institute admin for the install link, or use:

`https://<your-app-domain>/student/help/install`

Admin/teacher copy link from **Settings → Share student app install**.

---

## কেন phone-এ add করবেন?

- **এক ট্যাপে practice** — browser tab খুঁজতে হবে না
- **Exam reminder** — গুরুত্বপূর্ণ alert মিস কম হবে
- **অ্যাপের মতো** — দ্রুত খোলা যায়

---

## iPhone (Safari)

1. Link টি **Safari**-তে খুলুন (WhatsApp-এর ভেতরের browser নয়).
2. নিচে মাঝখানে **Share** (শেয়ার) বাটনে ট্যাপ করুন।
3. **Add to Home Screen** বেছে নিন।
4. উপরে ডানে **Add** চাপুন।
5. Home screen-এ নতুন icon দেখবেন — সেখান থেকে খুলুন।

**Push notification (iPhone):** আগে home screen-এ add করুন, তারপর app খুলে **Account → Browser push → On** করুন।

---

## Android (Chrome)

1. Link টি Chrome-এ খুলুন।
2. **Install app** বা **Add to Home screen** prompt এলে Confirm করুন।  
   না এলে: উপরে ডানে menu (⋮) → **Install app** / **Add to Home screen**।
3. Home screen থেকে icon টি খুলুন।

**Push notification:** **Account → Browser push → On** করুন।

---

## সমস্যা হলে

| সমস্যা | সমাধান |
| ------ | ------ |
| Install option নেই | Safari (iPhone) বা Chrome (Android) ব্যবহার করুন |
| Link WhatsApp-এ খুলছে | “Open in browser” / Safari-তে খুলুন |
| Login হচ্ছে না | Teacher-কে বলুন; password reset করুন |
| Push কাজ করছে না (iPhone) | আগে home screen-এ add করেছেন কিনা দেখুন |

আরও সাহায্য: app-এ **Get app** menu বা teacher-কে জানান।

## Teacher — ২ মিনিটের class script

> “সবাই phone-এ আমাদের practice app টি home screen-এ যোগ করবেন।  
> আমি link পাঠাচ্ছি / QR দেখাচ্ছি — খুলে **Add to Home Screen** করুন।  
> এরপর ওই icon থেকে practice শুরু করবেন।  
> Exam reminder চাইলে app খুলে **Account**-এ গিয়ে **Browser push** চালু করুন।  
> কোনো সমস্যা হলে আমাকে বলুন।”
